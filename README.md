

# API TEST AUTOMATION
Moxymind - Technical task #2
---
## Prerequisites

Please ensure the following tools are installed and configured: 

* [**Python** v3.13](https://www.python.org/downloads/release/python-3139/) or higher


* [**UV** package manager v0.9.9](https://docs.astral.sh/uv/getting-started/installation/) or higher
If installing from PyPI:
```bash
pip install uv==0.9.9
```

---
## Installation

1. **Clone the repository**:

```bash
git clone https://github.com/sulovj/api_test_automation.git
cd api_test_automation
```

2. **Install project dependencies**:

```bash
uv sync
```

---
## Run Tests
Execute tasks:

```bash
pytest tests/api
```

---
## Notes

* Report is generated and stored in root directory as 'report.html' file.
